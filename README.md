🩺 Disease Prediction Using Machine Learning
A Machine Learning project that predicts diseases based on patient symptoms using multiple classification algorithms. The system analyzes medical symptom data and predicts the most likely disease with high accuracy.

🚀 Features
Disease prediction using symptom data
Multiple Machine Learning models:
  Support Vector Machine (SVM)
  Gaussian Naive Bayes
  Random Forest Classifier
Data preprocessing and label encoding
Cross-validation for model evaluation
Accuracy and confusion matrix visualization
Ensemble prediction using combined model outputs

🛠️ Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn

📂 Dataset
The dataset contains:
Various symptoms as input features
Disease prognosis as target output
Files used:
Training.csv
Testing.csv

⚙️ Project Workflow
Load and preprocess medical dataset
Encode disease labels
Split dataset into training and testing sets
Train multiple classification models
Evaluate models using accuracy and cross-validation
Generate disease predictions from symptoms
Visualize performance using confusion matrix

📊 Models Used
Support Vector Machine (SVM)
Gaussian Naive Bayes
Random Forest Classifier
